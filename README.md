# Superstore Sales EDA

## Project Overview

This project presents an end-to-end Exploratory Data Analysis (EDA) of the Sample Superstore dataset using Python. The objective is to uncover business insights by analyzing sales performance, profitability, regional trends, customer purchasing behavior, and the impact of discount strategies.

The analysis includes data cleaning, statistical exploration, visualization, and business recommendations that can support data-driven decision-making.

---

## Business Objectives

- Analyze overall sales and profit performance.
- Identify the highest and lowest performing product categories.
- Compare sales across different regions.
- Evaluate the relationship between discounts and profitability.
- Detect outliers within sales data.
- Discover patterns using correlation analysis.
- Generate actionable business insights through visualization.
