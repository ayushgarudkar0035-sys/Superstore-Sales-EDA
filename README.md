# Superstore Sales EDA

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Sample Superstore dataset using Python.

## Objectives
- Understand sales trends
- Identify profitable and loss-making products
- Analyze sales by category and region
- Study the impact of discounts on profit
- Visualize important business insights

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Sample Superstore Dataset

## Analysis Performed
- Data Cleaning
- Missing Value Check
- Duplicate Check
- Descriptive Statistics
- Sales Analysis
- Profit Analysis
- Category Analysis
- Discount Analysis
- Correlation Analysis
- Outlier Detection
- Data Visualization

## Key Findings
- Technology generated the highest total sales.
- Technology also had the highest average sales.
- Office Supplies generated the lowest sales.
- High discounts generally reduced profit.
- A few products contributed significantly to overall profit, while others consistently generated losses.
